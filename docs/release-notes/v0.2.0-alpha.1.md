# terraform-google-betternat v0.2.0-alpha.1

Initial GCP alpha module packaging for BetterNAT.

This release wraps the `nowakeai/betternat` provider's
`betternat_gcp_gateway` resource for existing-VPC GCP validation. It targets
Firestore-backed BetterNAT HA, MIG-backed capacity repair, LoxiLB datapath
bootstrap, and optional stable public identity through an existing regional
static external IPv4 address.

Do not publish this module until the matching provider release and registry
validation are complete.
