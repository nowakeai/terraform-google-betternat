# terraform-google-betternat v0.2.1

Patch release for Terraform Registry example documentation.

## What Changed

- Added README files to every `examples/*` directory so Terraform Registry
  example pages describe the runnable examples instead of showing the default
  internal-submodule warning.
- Confirmed example-directory Terraform lockfiles are not part of the published
  module source. Users should generate their own lockfiles in their own root
  modules.

## Compatibility

No input, output, provider, or resource behavior changed.

## Validation

```sh
terraform fmt -check -recursive
terraform init -backend=false
terraform validate
```
